What is TxnGuard?

An end-to-end ML pipeline for real-world financial fraud detection with full per-prediction explainability.

Financial fraud causes billions in losses annually. TxnGuard is trained on over 1.75 million daily transactions spanning April–September 2018, covering three real fraud scenarios: high-value fraud, terminal compromise, and customer credential leakage.

The pipeline engineers nine predictive features across time, customer behaviour, and terminal risk. XGBoost handles class imbalance via scale_pos_weight and achieves ROC-AUC 0.9749 with only 438 false negatives after threshold tuning.

Every prediction uses real SHAP TreeExplainer values — not heuristics — to show exactly which features drove the fraud probability for that specific transaction.
